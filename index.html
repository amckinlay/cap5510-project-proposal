Title

Exploring sequence alignment visualizations

Team members

  • Andrew McKinlay

Abstract

Students learning bioinformatics for the first time often find difficulty rationalizing algorithms. Dynamic programming (DP) algorithms, for example, exhibit barriers for students trying to understand sequence comparison. This project aims to ease the transition into bioinformatics for students with a computer science and biology background alike. Web-based production and demonstration of bioinformatics algorithms through visualization and simulation will present an aid to students endeavoring into such topics. The project will focus on visualizations for heruistic, optimal, local, global, single, and multiple alignment algorithms, such as Needleman-Wunsch and Smith-Waterman to simulated annealing.

Plan of action

Before implementing the project, a review of existing bioinformatics visualizations will be conducted to find areas of improvement. A brief survey of non-bioinformatics, but related, algorithmic visualizations will accompany. The goal of the project is to create a set of visualizations that simulate the progression of particular bioinformatics algorithms, including their accompanying state. To accomplish this, visualizations, such as dot plots, density filters, graphs, and novel techniques will be combined to produce an overview of algorithm progression and performance. Visualizations, such as side-by-side representations of two different algorithms in synchronous progression, can give a visual understanding of comparative algorithm performance. Since the visualizations will be web-based, the project plans to implement a visualization framework, such as D3.js, to leverage HTML5 technologies, such as CSS transitions. Of course, the implementation language will be ECMAScript or a compile-to-JS language. Interactivity and quality will be prioritized over quantity of algorithm visualizations. The project plans to create a dynmaic environment for visualization. User input will be key to this end. The project does not plan to implement static animations (that is for YouTube). Hopefully, as the project progresses, more ideas will bubble up, creating a rich and interactive environment for learning basic bioinformatics algorithms.
